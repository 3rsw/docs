New Features

  • realtime extensions: CAmkES systems can now run on realtime seL4. It is possible to configure the realtime properties of systems built to run on realtime seL4.
    • For more details, see the Realtime Extensions section of the CAmkES Manual.
  • support for seL4 3.1.0
